# 👋 GitHub Profile README – Setup Guide & Template

## How to Get Your Profile README on GitHub

GitHub shows a special README on your profile page when you create a repository with **the same name as your GitHub username**.

### Steps to set it up:

1. Go to [github.com/new](https://github.com/new) and create a **new repository**.
2. Name it **exactly the same as your GitHub username** (e.g., `JesunAhmadUshno`).
3. Make sure it is set to **Public**.
4. Tick **"Add a README file"** and click **Create repository**.
5. Edit the `README.md` in that new repository – it will appear on your GitHub profile!

> **Tip:** You can copy the template below into that `README.md` and customise it.
